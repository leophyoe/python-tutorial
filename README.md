3.0 Python Variable




variable ဆိုသည်မှာ မည်သည့်ကိန်းဂဏန်းကိုမဆို အစားထိုးတွက်နိုင်သဖြင့် တန်ဖိုး ပြောင်းလဲနိုင်သော ကိန်းဂဏန်းကို ကိန်းရှင်( variable)လိုခေါ် ပါတယ်။
python programmingမှာ တော့ variableကို  data(value) တွေကိုအစားထိုးသိမ်းဆည်ပေးတယ် နေရာ အသုံးပြုပါတယ်။ mg mg ဆိုတယ် data(value)တစ်ခုကို
name ဆိုတယ် ကိန်းရှင်( variable) အစားထိုးသိမ်းဆည်မယ်ဆိုပါက equal သက်ေကိုအသုံးပြုပြီး
name =”mgmg”  name သည် mgmg နှင့်ညီသည် ဟု‌ရေးရပါမယ်။
Data တွေ ကို သိမ်းဆည် မယ်ဆိုရင် စာသား နှင့် ကိန်းဂဏန်း ဆိုပြီး (၂)မျိုးရှိတယ်။
(စာသားကို "string”လို ခေါ်ပြီး ကိန်းဂဏန်း ဆိုရင် ကိန်းပြည့် "Integer assignment” )
string (စာသားတန်ဖိုး)များ ကို ဖော်ပြမယ်ဆိုရင်
single quote (‘mg mg’)
Double quote (“mg”)
Triple quote  (“””mg”””)(‘’’mg’’’)
ဖြစ်ဖော်ပြလိုရပါတယ် ။
single quote သုံးသုံးdouple quote သုံးသုံး error မတတ်ပါဘူး ။
Triple quote က စာသာများကို တစ်ကြောင်းခြင်း multiline text ရေးမယ်ဆိုရင် triple quote ကိုအသုံးပြုပါတယ်(for example:1တွင်ကြည်ရန်)။
multiline text ရေးသားနည်း Triple quoteနဲ့ newline character ဆိုပြီး(၂)မျိုးရှိပါသည်
Triple quote က အပေါ်မှာကြည်လိုကါပါ။
newline character  က(\n)ကိုအသုံးပြုရပြီး စာ အကြောင်း တစ်ကြောင်း အဆုံးမှာ (n\)ထည်ရမည်ဖြစ်ပါသည် example >>>my name is phyoe hein kyaw \n I am 19 years old \n my live is yangon
ကိန်းဂဏန်များထည်သွင်း single quote/douple quote အသုံပြုစရာမလိုအပ်ပါ(for example:2 တွင်ကြည်ရန်)။
အခြား programming language များနှင့်မတူသည်မှာ Python သည် variable တစ်ခုကိုကြေငြာရန် command မများရေးရန်မလိုအပ်ဘူးပါဘူး။variable ထည်သွင်းရန် assignment operatorsက (=) ကို အသုံးပြုရပါတယ်။
